# 02-conditionals-if-else - Percabangan If/Else

## Overview
Modul ini mempraktikkan kontrol alur program menggunakan `if`, `else if`, dan `else`, termasuk penggunaan operator logika dan short statement untuk mengatur scope variabel secara aman.

## File Structure & Responsibilities
- `conditionals.go`: Entry point berisi `func main()` yang menunjukkan if-else standar, rantai if-else-if dengan `&&`/`||`, serta `if` dengan short statement.

## Key Concepts Learned
- Percabangan: `if`, `else if`, `else`
- Operator logika: AND (`&&`) dan OR (`||`)
- Short statement pada `if` dan dampaknya pada scope variabel

## How to Run
- `go run ./02-conditionals-if-else/conditionals.go`

## English Version

# 02-conditionals-if-else - If/Else Conditionals

### Overview
This module practices program flow control using `if`, `else if`, and `else`, including the use of logical operators and short statements to safely manage variable scope.

### File Structure & Responsibilities
- `conditionals.go`: The entry point containing `func main()` that demonstrates standard if-else, if-else-if chains with `&&`/`||`, and `if` with a short statement.

### Key Concepts Learned
- Conditionals: `if`, `else if`, `else`
- Logical operators: AND (`&&`) and OR (`||`)
- Short statement in `if` and its impact on variable scope

### How to Run
- `go run ./02-conditionals-if-else/conditionals.go`
