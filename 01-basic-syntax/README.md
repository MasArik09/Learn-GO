# 01-basic-syntax - Sintaks Dasar

## Overview
Modul ini memperkenalkan struktur program Go paling dasar, mulai dari `package main` hingga eksekusi melalui `func main()`.

## File Structure & Responsibilities
- `main.go`: Entry point berisi `func main()` yang mencetak output pertama dan contoh deklarasi variabel sederhana.

## Key Concepts Learned
- Struktur program dasar: `package main` dan `func main()`
- Import package standar (`fmt`) dan output ke terminal

## How to Run
- `go run ./01-basic-syntax/main.go`

# 01-basic-syntax - Basic Syntax

### Overview
This module introduces the most fundamental structure of a Go program, from `package main` to execution via `func main()`.

### File Structure & Responsibilities
- `main.go`: The entry point containing `func main()` that prints the first output and demonstrates simple variable declaration.

### Key Concepts Learned
- Basic program structure: `package main` and `func main()`
- Importing the standard package (`fmt`) and printing to the terminal

### How to Run
- `go run ./01-basic-syntax/main.go`
