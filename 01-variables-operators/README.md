# 01-variables-operators - Variabel & Operator

## Overview
Modul ini membangun fondasi representasi data di Go melalui deklarasi variabel, zero value, operasi aritmetika, dan type casting untuk menjaga presisi perhitungan.

## File Structure & Responsibilities
- `variables_operators.go`: Entry point berisi `func main()` yang mendemonstrasikan deklarasi variabel (eksplisit dan `:=`), zero value, serta operator aritmetika dan casting tipe.

## Key Concepts Learned
- Deklarasi variabel: `var` (eksplisit) vs short assignment `:=`
- Zero value pada tipe dasar
- Operator aritmetika (`+`, `-`, `*`, `%`) dan type casting (`float64(...)`) untuk pembagian

## How to Run
- `go run ./01-variables-operators/variables_operators.go`

# 01-variables-operators - Variables & Operators

### Overview
This module builds the foundation of data representation in Go through variable declaration, zero values, arithmetic operations, and type casting to maintain calculation precision.

### File Structure & Responsibilities
- `variables_operators.go`: The entry point containing `func main()` that demonstrates variable declaration (explicit and `:=`), zero values, as well as arithmetic operators and type casting.

### Key Concepts Learned
- Variable declaration: `var` (explicit) vs short assignment `:=`
- Zero values for basic types
- Arithmetic operators (`+`, `-`, `*`, `%`) and type casting (`float64(...)`) for division

### How to Run
- `go run ./01-variables-operators/variables_operators.go`
